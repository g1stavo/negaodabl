## Compatibility

- Linux
- Mac

## Installing
Install globally:

```shell
$ npm install -g negaodabl
```

## Run with [npx](https://www.npmjs.com/package/npx)

Default audio:

```shell
$ npx negaodabl
```

You can pass args to change the default audio (`apertaabraba`, `bagulho`).
Example:

```shell
$ npx negaodabl apertaabraba
```
